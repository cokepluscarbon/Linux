#31.温故而知新

###31.1 `cd -`返回上一次目录

```bash
[root@localhost share]# pwd
/usr/share
[root@localhost share]# cd /
[root@localhost /]# pwd
/
[root@localhost /]# cd -
/usr/share
[root@localhost share]# pwd
/usr/share
```
###31.2 `cd !$`把上个命令的参数作为cd参数使用

```bash
[root@localhost /]# pwd
/
[root@localhost /]# ls ~/Desktop/
jdk-8u111-linux-x64.tar.gz
[root@localhost /]# cd !$
cd ~/Desktop/
[root@localhost Desktop]# pwd
/root/Desktop
```
